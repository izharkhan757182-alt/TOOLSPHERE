How to use the Text to Binary Converter

  1. Step 1. Type plain text on the left to see it converted to binary (space-separated 8-bit bytes) on the right.
  2. Step 2. Or paste binary code on the right to decode it back to plain text on the left.
  3. Step 3. Both fields sync automatically as you type either one.

Frequently asked questions

What format is the binary?

Each character is shown as an 8-bit byte (e.g. 'A' becomes 01000001), separated by spaces.

Does this support special characters?

Yes, any UTF-8 character can be converted, though multi-byte characters will show as multiple 8-bit groups.
