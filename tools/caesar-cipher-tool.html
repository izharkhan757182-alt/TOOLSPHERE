Encode or decode text using a classic letter-shift cipher.

Encode / decode

How to use the Caesar Cipher Tool

  1. Step 1. Type the text you want to encode or decode.
  2. Step 2. Set a shift value (how many letters to shift, e.g. 3 means A becomes D).
  3. Step 3. Choose Encode to shift forward, or Decode to shift back — the result appears instantly.

Frequently asked questions

What is a Caesar cipher?

It's one of the earliest known encryption techniques, where each letter is shifted a fixed number of places in the alphabet. Julius Caesar reportedly used it for private correspondence.

Is this secure for real privacy?

No — Caesar ciphers are easily broken and meant for education or puzzles, not for protecting sensitive information.
