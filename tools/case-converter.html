How to use the Case Converter

Frequently asked questions

What is camelCase used for?

camelCase is commonly used for variable names in programming, joining words without spaces and capitalizing each word after the first.

What is snake_case used for?

snake_case is common in Python code and database column names — words are lowercase and joined with underscores.

Does this tool save my text?

No, all conversion happens locally in your browser.
