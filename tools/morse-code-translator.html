How to use the Morse Code Translator

  1. Step 1. Type text on the left to see it translated into Morse code (dots and dashes) on the right.
  2. Step 2. Or paste Morse code on the right — separate letters with spaces and words with a forward slash (/) — to decode it into text.
  3. Step 3. Both fields sync as you type.

Frequently asked questions

Is Morse code still used today?

Mostly for amateur radio, emergency signaling (like SOS), and educational purposes rather than everyday communication.

What symbols does this support?

Letters A-Z, numbers 0-9, and basic punctuation are supported.
