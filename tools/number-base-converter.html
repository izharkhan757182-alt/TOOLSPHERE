Frequently asked questions

Why do computers use binary?

Computer hardware is built from switches with two states (on/off), so binary (base 2) maps directly to how transistors and memory work.

Where is hexadecimal commonly used?

Hex is widely used for color codes, memory addresses, and byte-level data because it represents 4 binary digits compactly.
