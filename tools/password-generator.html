How to use the Password Generator

  1. Step 1. Set your desired password length with the slider.
  2. Step 2. Choose which character types to include: uppercase, lowercase, numbers, symbols.
  3. Step 3. Click Generate new password to create one, then copy it.
  4. Step 4. Use a different generated password for every account, and store them in a password manager.

Frequently asked questions

Is this password generator safe to use?

Yes. Password generation happens entirely in your browser using JavaScript's secure random number generator — nothing is transmitted or stored.

What makes a password strong?

Length matters more than complexity. Aim for at least 12–16 characters mixing letters, numbers, and symbols.

Should I reuse passwords?

No — use a unique password for every account so a breach on one site doesn't compromise others.
