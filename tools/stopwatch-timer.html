How to use the Stopwatch & Timer

  1. Step 1. Stopwatch: click Start to begin timing, Lap to record a split time, and Reset to clear.
  2. Step 2. Countdown timer: set minutes and seconds, then click Start — an alert sounds when it reaches zero.
  3. Step 3. Both tools keep running as long as this browser tab stays open.

Frequently asked questions

Does the timer work if I switch tabs?

Yes, it keeps counting in the background using the browser's clock, and will still alert you when it finishes.

Can I use decimal precision on the stopwatch?

Yes, the stopwatch displays down to hundredths of a second for precise lap timing.
